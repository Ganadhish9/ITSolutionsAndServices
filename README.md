# IT SOLUTIONS AND SERVICES - Online Course Platform

A premium, highly interactive React application for an IT education provider.

## Features
- **Modern UI/UX**: Dark mode aesthetic with glassmorphism and vibrant gradients.
- **Dynamic Animations**: Powered by `framer-motion` for smooth transitions and hover effects.
- **Course Catalog**: Data-driven course listing with category filters and "Buy Now" functionality.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **WhatsApp Integration**: Floating contact button for instant support.
- **Moving Banner**: Unique scrolling text banner for showcasing key skills.

## Tech Stack
- React (Vite)
- Framer Motion (Animations)
- Lucide React (Icons)
- Vanilla CSS (Custom Design System)

## How to Run

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```

3. **Build for Production**:
   ```bash
   npm run build
   ```

## Folder Structure
- `src/components`: Reusable UI components (Header, Footer, Hero, etc.)
- `src/styles`: CSS modules and global design system.
- `src/data`: Mock data for courses.
- `src/App.jsx`: Main application assembly.
